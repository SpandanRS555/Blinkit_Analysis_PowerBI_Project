# Blinkit_Analysis_PowerBI_Project

# 🛒 Blinkit Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes grocery sales data to uncover insights into product demand, outlet performance, and customer satisfaction. An interactive Power BI dashboard was developed to support data-driven decision-making.

---

## 🎯 Business Objective
To analyze:
- Sales performance  
- Customer preferences  
- Outlet efficiency  

---

## 🔍 Business Problem
The business aims to:
- Understand which products drive the most revenue  
- Evaluate outlet performance across different locations and sizes  
- Improve customer satisfaction using rating analysis  
- Optimize inventory and operational strategies  

---

## 📂 Dataset
- ~8,500 records  
- 12 features including:
  - Item Type, Fat Content  
  - Outlet Size, Location, Type  
  - Sales, Ratings  

---

## 📊 Key Analysis

### 🔹 KPIs
- Total Sales  
- Average Sales  
- Number of Items Sold  
- Average Rating  

---

### 🔹 Product Analysis
- Sales by Fat Content (Low Fat vs Regular)  
- Sales by Item Type  
- Multi-metric comparison:
  - Sales  
  - Number of Items  
  - Ratings  

---

### 🔹 Outlet Analysis
- Sales by Outlet Establishment Year  
- Performance by Outlet Size  
- Tier-wise location analysis (Tier 1, 2, 3)  
- Outlet type comparison  

---

## 🛠️ Tools & Technologies
- Power BI  
- Power Query  
- DAX (Data Analysis Expressions)  

---

## ⚙️ Data Processing
- Cleaned inconsistent values (e.g., LF → Low Fat, Reg → Regular)  
- Handled missing values and validated data quality  
- Created calculated measures using DAX:
  - Total Sales (SUM)
  - Average Sales (AVERAGE)
  - Number of Items (COUNTROWS)
  - Average Rating  

---

## 📊 Dashboard Features
- Interactive filters (Item Type, Outlet Type, Location)  
- Dynamic KPI cards  
- Metric switching (Sales, Avg Sales, Items, Ratings)  
- Visualizations:
  - Donut Chart (Fat Content)
  - Bar Chart (Item Type)
  - Line Chart (Outlet Trend)
  - Matrix Table (Outlet Comparison)

---

## 📈 Key Insights
- Identified top-performing product categories  
- Highlighted differences in outlet performance across city tiers  
- Revealed opportunities to improve customer satisfaction  
- Showed impact of outlet size and type on sales  

---

## 🚀 Business Impact
- Supports inventory optimization  
- Enables strategic expansion planning  
- Improves operational efficiency  
- Helps stakeholders make data-driven decisions
